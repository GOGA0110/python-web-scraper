# Python Web Scraper

A simple Python script for collecting data from websites.

## Features

* collects company names
* collects email addresses
* saves data to a CSV file

## Technologies

* Python
* requests
* BeautifulSoup
* CSV

## How to run

1. Install required libraries:

pip install requests beautifulsoup4

2. Run the script:

python scraper.py
